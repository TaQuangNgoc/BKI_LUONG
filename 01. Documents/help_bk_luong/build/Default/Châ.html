Chấm công từ excel
Nhập liệu > Chấm công từ excel

 

Bạn có thể làm việc với các danh mục sau:

I. Theo Thời gian

Bạn chọn tháng  cần chấm công tại  mục "tháng". Sau đó, nếu bạn muốn:

1. xem dữ liệu đã được cập nhật

Nhấn  nút hiển thị dữ liệu đã được cập nhật , muốn thoát nhấn nút "thoát"

2. Upload dữ liệu chấm công

 

Nhấn vào nút upload dữ liệu chấm công

Chọn file cần thao tác, nhấn Open

3. Tải file excel mẫu

 

Chọn nơi lưu file excel, nhấn save

Mở file ra và điền các thông tin tương ứng

=>Nhấn nút "Lưu chấm công"

 

 

II. Làm thêm theo thời gian

 

Bạn chọn tháng  cần chấm công tại  mục "tháng", chọn loại làm thêm tại mục "chọn loại làm thêm".Sau đó, nếu bạn muốn:

1. Xem dữ liệu đã được cập nhật

 

Nhấn  nút hiển thị dữ liệu đã được cập nhật , muốn thoát nhấn nút "thoát"

2. Upload dữ liệu chấm công

Nhấn vào nút upload dữ liệu chấm công

Chọn file cần thao tác, nhấn Open

3. Tải file excel mẫu

Chọn nơi lưu file excel, nhấn save

Mở file ra và điền các thông tin tương ứng

=>Nhấn nút "Lưu chấm công"

IV. Các loại tiền khác

 

 

Bạn chọn tháng  cần chấm công tại  mục "tháng" và  chon loại tiền tại mục"chọn loại tiền". Sau đó, nếu bạn muốn:

1. xem dữ liệu đã được cập nhật

Nhấn  nút hiển thị dữ liệu đã được cập nhật , muốn thoát nhấn nút "thoát"

Nếu bạn muốn xóa dữ liệu nào đó, chọn dòng cần xóa trên form, nhấn nút "Xóa dòng",form hiện lên, bạn nhấn "Yes "để xóa, ngược lại nhấn "No"

2. Upload dữ liệu chấm công

Nhấn vào nút upload dữ liệu chấm công

Chọn file cần thao tác, nhấn Open

3. Tải file excel mẫu

Chọn nơi lưu file excel, nhấn save

Mở file ra và điền các thông tin tương ứng

=>Nhấn nút "Lưu chấm công"
